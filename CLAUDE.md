# CLAUDE.md — vinsonconsulting

## Storage rules (Vinson standard)
- This repo lives under ~/GitHub/. Never move, copy, or clone it into
  ~/SynologyDrive, /Volumes, ~/Library/CloudStorage, or any synced path.
- Drive reference material: read from /Volumes/drive-mirror/<path> (read-only)
  or ask me to attach files. Never write to /Volumes.
- Working outputs go to ./out/ (gitignored) or committed paths.
- Deliverables bound for Drive: as a final explicit step, copy to
  ~/SynologyDrive/<client>/deliverables/ — never use that location as the
  working directory.
- Never edit files in place on /Volumes or ~/SynologyDrive from this repo's
  session; copy-in → edit → copy-out.
